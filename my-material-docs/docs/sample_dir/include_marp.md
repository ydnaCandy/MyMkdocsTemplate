# Marp スライドの表示

<iframe 
  # 💡 重要な点: MkDocsのMarkdownファイルから見て、HTMLファイルがどこにあるかを指定します。
  # 今回、MarkdownファイルとHTMLファイルが同じディレクトリ構造（./docs/sample_dir/）にあると仮定しています。
  src="../slide.html"  
  
  width="100%" 
  height="600" 
  allowfullscreen
  
  # スライド操作のためのスクロールバーを非表示にし、境界線を目立たせます
  scrolling="no"
  style="border: 2px solid #3f51b5; border-radius: 5px;">
</iframe>

## スライドの操作方法

埋め込まれたスライドをクリックした後、キーボードの矢印キー（→、←）でページを切り替えることができます。
